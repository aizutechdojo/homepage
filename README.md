# 東北TECH道場 会津道場

東北Tech道場は、アプリ作成に興味がある人が集まる道場です。
一緒にアプリを作って勉強しながら、教えあうなどモチベーションを高めることができる場です。月一回ほど集まり、自分の興味のある分野でアプリやサービスをもくもく会形式で作成する会です。現役エンジニアも参加しているので、ぜひいろいろきいてみてください。
